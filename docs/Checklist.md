# YoScore MVP Development Checklist

## Step 1: Backend Foundation
- [ ] `feature/data-model` → Database schema created
- [ ] `feature/backend-api` → API endpoints implemented
- [ ] `feature/auth-security` → JWT, RBAC, password hashing secured

## Step 2: Scoring & Proctoring
- [ ] `feature/scoring-engine` → Challenge scoring & trust score calculation implemented
- [ ] `feature/proctoring` → Camera + browser monitoring implemented

## Step 3: Frontend & Dashboard
- [ ] `feature/frontend-ui` → Challenge selection, code editor, submission flow implemented
- [ ] `feature/dashboard` → Dashboard with scores & history implemented
- [ ] `feature/reference-panel` → Reference docs access implemented

## Integration & Testing
- [ ] Unit tests for backend & scoring engine
- [ ] Frontend-backend integration tests completed

## MVP Launch
- [ ] Merge `dev` → `main`
- [ ] Tag release `v0.1-MVP`
- [ ] Deploy on test server / environment
