# Print Checklist

## Functional Checks
- [ ] All five phases are present and ordered correctly.
- [ ] Phase 1 includes all 11 required subsections.
- [ ] Phase 2 includes all six required diagram categories.
- [ ] Internal anchor links in the table of contents navigate correctly.
- [ ] Tables and SQL code blocks are visible and readable.

## Diagram Checks
- [ ] Use case diagram appears in Phase 1.
- [ ] Class, Sequence, State, Activity, Component, and Deployment diagrams appear in Phase 2.
- [ ] Every diagram has a figure caption.
- [ ] No diagram appears stretched or blurred.

## Print/PDF Checks
- [ ] Print preview uses A4 pages.
- [ ] Navigation section is hidden in print mode.
- [ ] Each phase starts on a new page.
- [ ] Headings are not left orphaned at the bottom of pages.
- [ ] Figures are not cut across pages.
- [ ] Final PDF is legible in grayscale.

## Submission Packaging Checks
- [ ] PDF exported and named correctly.
- [ ] DOCX exported and matches PDF content.
- [ ] Source files (HTML/CSS/assets) included.
- [ ] Supporting docs (SQL/testing/manuals/reports) included.
