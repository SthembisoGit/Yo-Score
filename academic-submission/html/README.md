# Academic HTML Package Guide

## What This Package Contains
- `index.html`: single master academic document covering Phases 1-5.
- `styles.css`: formal academic and print-friendly styling.
- `assets/`: UML diagram PNG files referenced by `index.html`.
- `print-checklist.md`: pre-export quality checklist.

## Open Locally
1. Open `academic-submission/html/index.html` in Chrome or Microsoft Edge.
2. Confirm all images load (no broken figure icons).
3. Review section navigation links at the top.

## Export to PDF (Chrome/Edge)
1. Press `Ctrl+P`.
2. Destination: `Save as PDF`.
3. Layout: `Portrait`.
4. Paper size: `A4`.
5. Margins: `Default` (or `Custom` if your lecturer provided exact margins).
6. Scale: `Default` (100%).
7. Enable `Background graphics` only if required by your institution.
8. Save as `YoScore-Academic-Submission.pdf`.

## DOCX Workflow (Free)
1. Open `index.html` in LibreOffice Writer.
2. Verify heading levels and figure captions.
3. Save as `.docx`.
4. Export the same file to PDF for consistency.

## Recommended Final Bundle
- Exported PDF
- DOCX file
- `academic-submission/` folder (source markdown, diagrams, SQL, testing docs)
- Full project source zip
- Database DDL and backup files
