# AI Prompt Kit (Free Tools)

Use these prompts in any free LLM (ChatGPT free, Gemini free, Copilot free tier).

## 1) Proposal Refinement Prompt
```
I am submitting an Industry Project proposal for a 3-tier system.
Please improve this section for academic clarity without changing technical meaning:
[PASTE SECTION]
Constraints:
- Keep formal academic tone.
- Keep headings and numbering.
- Keep it realistic and auditable.
```

## 2) SMART Objectives Review Prompt
```
Review these project objectives and rewrite them so each is SMART
(Specific, Measurable, Achievable, Relevant, Time-bound):
[PASTE OBJECTIVES]
Return as numbered list with measurable targets.
```

## 3) Heuristic Evaluation Assistant Prompt
```
I am evaluating a web app using Nielsen heuristics.
Given this UI description and observed issues, produce a concise table with:
Heuristic, Issue, Severity (Low/Medium/High), Impact, Recommendation.
[PASTE OBSERVATIONS]
```

## 4) Test Case Quality Prompt
```
Improve these test cases so they are clear and executable:
[PASTE TEST CASES]
For each test include Preconditions, Steps, Expected Result.
```

## 5) UML Review Prompt
```
Review this PlantUML diagram for correctness and clarity:
[PASTE PUML]
Return:
1) Detected modeling problems
2) Improved PlantUML code
```
