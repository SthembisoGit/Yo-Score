# Academic Submission Package

This folder contains the lecturer-facing submission material for YoScore, including the phased
write-up, testing documents, diagrams, and the printable HTML/PDF export package.

## Key areas

- `00-Index.md`: package entry point
- `Phase-*.md`: phase-by-phase academic deliverables
- `testing/`: test plan and test case documentation
- `diagrams/`: exported UML and supporting visuals
- `html/`: printable academic submission build

Use this folder when preparing the formal academic hand-in rather than the runtime application.
